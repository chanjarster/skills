# 文档 CRUD

本文档涵盖文档的创建、更新和删除操作。

所有操作都必须同步更新索引，参见 [5-索引维护规则.md](5-索引维护规则.md)。

## 创建文档

创建新文档时：

1. 确定文档类型，使用对应的模板（见 [SKILL.md 文档类型指南](../../SKILL.md#文档类型指南)）
2. 放置到正确的路径，遵循命名规范
3. 更新所在层级的 `0-索引.md`

**命名规范**：

- 需求文档/操作手册：`<序号>-<业务领域>.md`（操作手册与需求文档的序号和领域名保持一致）
- 代码地图：`<序号>-<业务领域>.md`，公共组件为 `0-公共组件.md`
- 任务目录：`<yyMMdd>-<HHmm>-<任务标题>`
- 示例文档：`<序号>-<示例名称>.md`

## 更新文档

1. 先读取现有内容
2. 在对应章节增量更新，保持文档精简
3. 更新所在层级的 `0-索引.md`

## 删除文档

删除文档时：

1. 确认要删除的文档路径
2. 删除文档文件或目录
3. 更新所在层级的 `0-索引.md`
4. 若删除的文档被其他文档引用，检查并清理相关引用
