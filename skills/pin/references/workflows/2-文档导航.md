# 文档导航（搜索）

Pin 使用 `0-索引.md` 作为每层目录的导航入口。**永远优先读取索引文件来定位目标文档**，而不是扫描整个目录。

导航流程：

1. 先用 glob 搜索所有 `pin-docs/*/0-索引.md`（`*` 代表文档类型），了解当前有哪些文档类型目录，根据用户意图选择要读取的文档类型目录
2. 读取目标目录的 `0-索引.md` 了解有哪些文档可用
3. 根据用户意图匹配摘要，找到对应文档路径
4. 读取目标文档

例如搜索"用户登录代码设计"相关文档：

- 搜索 `0-索引.md`，找到 `0-业务知识/`、`1-技术文档-模块A/`、`2-UI设计/` 这些文档类型目录，根据用户搜索意图，决定进一步对 `1-技术文档-模块A/` 进行搜索
- 读取 `1-技术文档-模块A/0-索引.md` 找到相关文档
- 最后读取具体文档内容
